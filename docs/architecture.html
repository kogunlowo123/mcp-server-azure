mcp-server-azure

MCP server providing tools for Azure VM, Storage, Key Vault, Resource Groups, AKS, and Monitor operations

TypeScript 5.3+ MCP Server v1.0.0

Executive Summary

An MCP (Model Context Protocol) server that exposes Azure cloud operations as tools for AI assistants. Built with the official MCP SDK and Azure SDK for JavaScript, it provides six core tools covering Virtual Machines, Blob Storage, Key Vault secrets, Resource Groups, AKS clusters, and Azure Monitor metrics. The server communicates via stdio transport, enabling integration with any MCP-compatible client including Claude Desktop. Authentication supports both Azure AD service principal credentials and Azure CLI authentication.

Data Flow / Request Flow

  1. Client Request -- An MCP client (e.g., Claude Desktop) sends a JSON-RPC tool invocation over the stdio transport.
  2. Tool Resolution -- The MCP server matches the tool name and validates parameters using Zod schemas.
  3. Azure Authentication -- DefaultAzureCredential resolves credentials (service principal env vars or CLI token).
  4. Service API Call -- The tool handler uses the appropriate Azure SDK client to call the Azure REST API.
  5. Response Formatting -- Results are serialized and returned to the MCP client as structured content.

Production Readiness Checklist

  • Configure Azure AD service principal or managed identity with least-privilege RBAC
  • Set AZURE_SUBSCRIPTION_ID and authentication environment variables
  • Build TypeScript to JavaScript (npm run build)
  • Test all six tools against target Azure subscription
  • Configure MCP client (Claude Desktop) with server entry point
  • Set up Key Vault access policies for secret retrieval
  • Deploy as Docker container for production use
  • Enable Azure Monitor diagnostic logging for audit trail

Deployment

Local Development

git clone https://github.com/kogunlowo123/mcp-server-azure.git
cd mcp-server-azure
npm install
npm run build
export AZURE_SUBSCRIPTION_ID=xxx
npm start

Docker / Production

docker build -t mcp-server-azure .
docker run \
  -e AZURE_SUBSCRIPTION_ID=xxx \
  -e AZURE_TENANT_ID=xxx \
  -e AZURE_CLIENT_ID=xxx \
  -e AZURE_CLIENT_SECRET=xxx \
  mcp-server-azure
