# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial release of mcp-server-azure.
- Tool: `vm_list` - List Azure virtual machines.
- Tool: `storage_list_blobs` - List blobs in an Azure Storage container.
- Tool: `keyvault_get_secret` - Retrieve a secret from Azure Key Vault.
- Tool: `resource_group_list` - List Azure resource groups.
- Tool: `aks_list_clusters` - List AKS clusters in a subscription.
- Tool: `monitor_list_metrics` - List Azure Monitor metrics for a resource.
- MCP server with stdio transport.
- Docker support.
